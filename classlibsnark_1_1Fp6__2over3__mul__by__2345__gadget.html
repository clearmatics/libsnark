Detailed Description

template<typename Fp6T>
class libsnark::Fp6_2over3_mul_by_2345_gadget< Fp6T >

Gadget that creates constraints for Fp6 multiplication by a Fp6 element B for which B.c0.c0 = B.c0.c1 = 0.
